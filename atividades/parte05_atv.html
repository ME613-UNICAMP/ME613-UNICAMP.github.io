Atividade 05 - COVID-19 na Regio de Campinas e Estado de SP

O conjunto de dados spdata presente no arquivo spdata.rda possui informaes acerca do nmero acumulado de casos e bitos confirmados em trs diferentes nveis:

  • Estado de SP
  • Regio Metropolitana de Campinas (RMC)
  • Regio Metropolitana de Campinas (RMC) + Piracicaba + Limeira: municpios atendidos por campi da UNICAMP
  • Municpios da RMC

Para carregar os dados, voc precisar utilizar o pacote tidyverse, como sugerido abaixo:

library(tidyverse)
load("../dados/spdata.rda")
date city place_type confirmed deaths estimated_population_2019

2020-02-26 SP (Estado) state 1 0 45919049
2020-02-27 SP (Estado) state 1 0 45919049
2020-02-28 SP (Estado) state 2 0 45919049
2020-02-29 SP (Estado) state 2 0 45919049
2020-03-01 SP (Estado) state 2 0 45919049

Para obter dados de um municpio, voc pode aplicar filtros utilizando a coluna city. Por exemplo, para os casos e bitos ocorridos em Jundia:

spdata %>% filter(city == "Jundia") %>% head() %>%  knitr::kable()
date city place_type confirmed deaths estimated_population_2019
2020-03-25 Jundia city 1 0 418962
2020-03-26 Jundia city 2 0 418962
2020-03-27 Jundia city 3 0 418962
2020-03-30 Jundia city 3 0 418962
2020-03-31 Jundia city 3 0 418962
2020-04-01 Jundia city 5 0 418962

Estas informaes podem ser utilizadas para compor os grficos de casos e bitos acumulados:

library(ggplot2)
spdata %>% filter(city == "Jundia") %>% 
  ggplot(aes(date, confirmed)) + geom_point() + theme_bw() +
  xlab("Data") + ylab("Casos confirmados at a data") + ggtitle("Casos Confirmados")

spdata %>% filter(city == "Jundia") %>% 
  ggplot(aes(date, deaths)) + geom_point() + theme_bw() +
  xlab("Data") + ylab("bitos confirmados at a data") + ggtitle("bitos Confirmados")
